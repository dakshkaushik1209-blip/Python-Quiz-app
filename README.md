# AI Quiz Application 

A simple command line quiz application built using Python. This project asks multiple choice questions related to Generative AI, checks user answers, calculates the score, and displays the final percentage.

## Features

- Multiple-choice quiz system
- 10 Generative AI questions
- Input validation (A, B, C, D only)
- Score tracking
- Correct and Wrong answer counter
- Percentage calculation
- Displays correct answer when the user is wrong

## Technologies Used

- Python
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Input Validation

## How to Run

```bash
python quiz_app.py
```

## Example

```text
1. What is the primary goal of Generative AI?

A) To categorize and label existing data
B) To create new, original content by learning from existing data patterns
C) To perform complex mathematical calculations faster than a calculator
D) To strictly automate repetitive physical tasks on assembly lines

Enter your answer :- B

Correct
```
